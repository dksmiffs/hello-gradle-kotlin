This project demonstrates basic [Kotlin][1] with [Gradle][2]'s Kotlin DSL.

## Usage

1.  [Install Gradle][3].
2.  From bash on Linux, or PowerShell or Git Bash on Windows, type `./gradlew` from the top level of this project.
